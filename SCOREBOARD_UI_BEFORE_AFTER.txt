BUTTON 1: ✅ Save Match Result (GREEN)
──────────────────────────────────────────────────────────────────────────────

  Color: Green gradient (from-green-500 to-emerald-500)
  Icon: ✅ (CheckCircle)

  When clicked:
    1. Button shows loading state: "💾 Saving match result..."
    2. Data sent to backend API
    3. Backend saves to database with:
       - Winner ID
       - Scores (Ippon, Waza-ari, Yuko, Shido)
       - Win method
       - Match logs/events
    4. Next-round match auto-updated with winner
    5. Shows: "✅ Match result saved to database!"
    6. Message disappears after 2 seconds

  Database Entry Created:
    ┌─────────────────────────────────────────┐
    │ matchId: "M1_Q_48kg_FEMALE"             │
    │ status: "COMPLETED"                     │
    │ winnerId: "player_123"                  │
    │ scoreA: { ippon: 1, wazaAri: 0, ... }   │
    │ scoreB: { ippon: 0, wazaAri: 0, ... }   │
    │ winMethod: "Decision"                   │
    │ logs: [...match events...]              │
    │ Timestamp: 2026-06-10T15:45:00          │
    └─────────────────────────────────────────┘

  Auto-Saved Even Without Click?
    YES! The data auto-saves in the background.
    This button is just for USER CONFIRMATION.


BUTTON 2: 📥 Download Report (PDF) (BLUE)
──────────────────────────────────────────────────────────────────────────────

  Color: Blue gradient (from-blue-500 to-cyan-500)
  Icon: 📥 (Download)

  When clicked:
    1. Browser generates professional PDF
    2. Print dialog opens
    3. User can:
       a) Select printer → Print to paper
       b) "Save as PDF" → Save to computer
       c) "Cancel" → Close dialog

  PDF Contains:
    ├─ Tournament Title & Info
    ├─ Match Details (Mat #, Match #)
    ├─ Winner Card
    │  ├─ Name: MANIKANDAN M
    │  ├─ Club: Yeotmal Club
    │  └─ Scores: Ippon, Waza-ari, Yuko, Shido
    │
    ├─ Opponent Card
    │  ├─ Name: SIDDHI SHARMA
    │  ├─ Club: Mumbai Club
    │  └─ Scores: Ippon, Waza-ari, Yuko, Shido
    │
    ├─ Decision: "Victory by Decision"
    ├─ Match Events Log
    └─ Timestamp & TNJA Footer

  File Information:
    - Format: PDF (printable)
    - Size: ~100-150 KB
    - Compatible with: All browsers, Adobe, etc.
    - Can be: Printed, Saved, Emailed, Archived


BUTTON 3: ↺ New Match / Reset (GRAY)
──────────────────────────────────────────────────────────────────────────────

  Color: White/gray (white/10 hover white/20)
  Icon: ↺ (RotateCcw)

  When clicked:
    1. Clears all scores
    2. Resets match timer
    3. Clears logs and events
    4. Removes winner declaration
    5. Saves empty state to DB
    6. Ready for next match

  What Gets Reset:
    - Winner: null
    - ScoreA: { ippon: 0, wazaAri: 0, yuko: 0, shido: 0 }
    - ScoreB: { ippon: 0, wazaAri: 0, yuko: 0, shido: 0 }
    - Win Method: ""
    - Logs: []
    - Timer: Resets to match duration
    - Golden Score: false
