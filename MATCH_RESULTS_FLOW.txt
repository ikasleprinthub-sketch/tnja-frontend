                           KEY FEATURES
═══════════════════════════════════════════════════════════════════════════════

✅ AUTO-ADVANCEMENT
   When match completes → Winner auto-advances to next round
   Next-round "TBD" becomes winner's name with animation ✨

✅ NEXT MATCH VISIBILITY
   For each winner, see:
   - Which round they advance to
   - Match number and mat number
   - Who their opponent is (or waiting status)

✅ RESULTS TRACKING
   View all completed matches in one place
   Filter by age group, gender, weight category

✅ PDF REPORTS
   Professional match records for:
   - Tournament information
   - Winner & loser details
   - Seeding information
   - Next match destination
   - Print or save as PDF

✅ ANIMATIONS & UI
   Smooth transitions when viewing results
   Color-coded: Green (winner), Gray (loser), Blue (next match)
   Responsive design on all devices

                         USAGE EXAMPLE
═══════════════════════════════════════════════════════════════════════════════

SCENARIO: Admin running 48kg Women's Tournament with 8 players

1. Create tournament → 48kg, FEMALE, 8 players register
2. Generate draw → IJF seeding positions assigned
3. Run matches:
   - Match#1: SIDDHI beats PRIYA → advances to QF#7
   - Match#2: SANSKRUTI beats FATIMA → advances to QF#7
   - Match#3: ANJALI beats NEHA → advances to QF#8
   - Match#4: PREETI beats MAHIMA → advances to QF#8
4. Click Results Tab → See all 4 completed matches
5. Click Export PDF for each match → Save match records
6. Continue with QF#7 between SIDDHI vs SANSKRUTI
7. Continue with QF#8 between ANJALI vs PREETI
8. Winners advance to semifinals automatically
